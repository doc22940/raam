> **raam** (estonian) ˈrɑːm _**n**._ frame.

Painfully boring cosmetic-free React.js components for structure and layout.

## Components

### Inline

<Inline>
  {Array.from({ length: 24 }).map((item, i) => (
    <Box
      key={i}
      sx={{ border: "1px solid black", width: "2rem", height: "2rem" }}
    />
  ))}
</Inline>

> Caveats
>
> As of April 2020 flex `gap` is only supported by Firefox. As a fallback,
> negative margins are used on the outer compoonent to counteract margins inside.
> If you place an `Inline` component it will **not** affect the margin flow, but
> you may have issues if you want to adjust it.
>
> In a case where you want to adjust the margin, you should wrap `Inline` with
> another element.

#### List

By default, all `Inline` child elements render wrapped in a `div`.
If a list element is defined via the `as` prop (e.g. `ul` or `ol`), all items
will render wrapped in an `li` without a list-style-type.

<Inline as="ul">
  {Array.from({ length: 24 }, (v, k) => k + 1).map(item => (
    <ThemeUILink key={item} href={`#list-item-${item}`}>
      Item {item}
    </ThemeUILink>
  ))}
</Inline>
